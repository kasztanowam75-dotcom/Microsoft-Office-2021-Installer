Komendy:
cd C:\Office

setup.exe /configure Configuration.xml

Aktywacja Office za darmo:
- Otwórz plik OmniKey.exe 
- Następnie kliknij w oknie Tak (aby uruchomić program jako administrator)
- w oknie naciśnij klawisz Y i kliknij Enter
- Poczekaj aż wyskoczy okno z opcjami 1, 2, 3 itd.
- Wybierz Ohook (kliknij 2)
- Kliknij po tym klawisz 1 aby aktywować Office za darmo
Brawo gdy program napisze Press any key to Go back... zamknij okno i aplikację OmniKey
